# FolderTree Component Documentation

## Overview

The `FolderTree` component is a customizable folder structure UI component that allows users to represent and interact with a hierarchical file system.

## Props

### FolderTree Props

| Prop         | Type     | Default | Description                         |
| ------------ | -------- | ------- | ----------------------------------- |
| `className`  | `string` | `-`     | Additional classes for styling      |
| `indentSize` | `number` | `16`    | Indentation size for nested folders |

### Folder Props

| Prop          | Type      | Default | Description                          |
| ------------- | --------- | ------- | ------------------------------------ |
| `element`     | `string`  | `-`     | Name of the folder                   |
| `defaultOpen` | `boolean` | `false` | Determines if the folder starts open |
| `className`   | `string`  | `-`     | Additional classes for styling       |

### File Props

| Prop        | Type     | Default | Description                    |
| ----------- | -------- | ------- | ------------------------------ |
| `className` | `string` | `-`     | Additional classes for styling |

## Usage Steps

1. **Wrap files and folders inside the `FolderTree` component.**
2. **Use the `Folder` component to create a collapsible directory.**
3. **Place `File` components inside `Folder` to represent files.**
4. **Use `defaultOpen` to control initial visibility.**

### Example

```tsx
<FolderTree className="bg-gray-100 p-4">
  <Folder element="src" defaultOpen={true}>
    <Folder element="components" defaultOpen={true}>
      <File>
        <p>Button.tsx</p>
      </File>
    </Folder>
    <File>
      <p>index.tsx</p>
    </File>
  </Folder>
</FolderTree>
```
