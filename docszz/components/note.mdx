# Note Component

The `Note` component is used to display contextual messages with different styles based on the type of note.

## Props

| Prop        | Type                                                           | Default  | Description                       |
| ----------- | -------------------------------------------------------------- | -------- | --------------------------------- |
| `type`      | `none` \| `info` \| `warning` \| `alert` \| `success` \| `tip` | `'none'` | Determines the style of the note. |
| `className` | `string`                                                       | `''`     | Additional CSS classes.           |
| `hideIcon`  | `boolean`                                                      | `false`  | Hides the note icon when `true`.  |
| `children`  | `ReactNode`                                                    | `-`      | The content of the note.          |

## Usage Steps

1. Import the `Note` component from your components folder.
2. Use the `type` prop to specify the type of note.
3. Optionally, use `hideIcon` to hide the icon.
4. Add your message inside the `<Note>` component.

### Example

```tsx
<Note type="success" hideIcon={true}>
  Success without an icon!
</Note>
```

This component helps in displaying contextual messages effectively in your UI.
