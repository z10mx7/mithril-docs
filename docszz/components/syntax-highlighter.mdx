# Syntax Highlighter Component Documentation

The **Syntax Highlighter** component provides a customizable and interactive way to display code snippets with syntax highlighting. It supports multiple tabs for different code variations and includes a copy-to-clipboard feature.

## Props

| Prop             | Type                                                    | Default           | Description                                                                            |
| ---------------- | ------------------------------------------------------- | ----------------- | -------------------------------------------------------------------------------------- |
| `tabs`           | `Record<string, { syntax: string; language?: string }>` | `{}`              | An object where keys represent tab names and values contain code and language details. |
| `className`      | `string`                                                | `undefined`       | Custom CSS classes for additional styling.                                             |
| `themeMode`      | `'light' \| 'dark'`                                     | `'dark'`          | Controls the theme mode of the syntax highlighter.                                     |
| `lightTheme`     | `Record<string, any>`                                   | `oneLight`        | Custom theme for light mode.                                                           |
| `darkTheme`      | `Record<string, any>`                                   | `oneDark`         | Custom theme for dark mode.                                                            |
| `indicatorColor` | `string`                                                | `'bg-yellow-500'` | Custom color for the active tab indicator.                                             |
