## Props

| Prop        | Type                      | Default | Description                             |
| ----------- | ------------------------- | ------- | --------------------------------------- |
| `className` | `string`                  | `''`    | Additional CSS classes.                 |
| `open`      | `boolean`                 | `false` | Controls the visibility of the menu.    |
| `setOpen`   | `(open: boolean) => void` | `-`     | Function to toggle the menu state.      |
| `children`  | `ReactNode`               | `-`     | The content inside the menu components. |

## Usage Steps

1. Import the `Menu`, `MenuTrigger`, `PopMenu`, and `MenuItem` components.
2. Wrap the menu items inside the `Menu` component.
3. Use `MenuTrigger` to create a clickable button for toggling the menu.
4. Wrap menu items inside `PopMenu` for display.
5. Optionally, include an input field inside `PopMenu` to create an autocomplete menu.
