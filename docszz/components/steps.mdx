# Steps Component Documentation

The **Steps** component provides a structured way to represent a multi-step process. It supports both horizontal and vertical layouts, making it ideal for onboarding flows, wizards, and process visualizations.

## Props

| Prop               | Type                 | Default   | Description                                                |
| ------------------ | -------------------- | --------- | ---------------------------------------------------------- |
| `contentPosition`  | `"right" \| "below"` | `"below"` | Defines whether the content appears to the right or below. |
| `className`        | `string`             | `""`      | Custom class for styling the steps container.              |
| `titleClassName`   | `string`             | `""`      | Custom class for styling the step title.                   |
| `contentClassName` | `string`             | `""`      | Custom class for styling the step content.                 |
| `lineClassName`    | `string`             | `""`      | Custom class for styling the connector line.               |
| `numberClassName`  | `string`             | `""`      | Custom class for styling the step number circle.           |

The **Steps** component is highly flexible and allows for styling customization to fit various UI requirements.
