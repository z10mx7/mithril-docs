# Tabs Component Documentation

The **Tabs** component is a customizable tab system that supports multiple sizes and provides a smooth transition effect. It is designed for modern, theme-aware applications that use CSS variables (defined in your global CSS) for light and dark modes.

## Props

| Prop            | Type                                   | Default     | Description                                       |
| --------------- | -------------------------------------- | ----------- | ------------------------------------------------- |
| `defaultValue`  | `string`                               | `""`        | The initial active tab when the component mounts. |
| `value`         | `string`                               | `""`        | The controlled value of the active tab.           |
| `onValueChange` | `(value: string) => void`              | `undefined` | Callback function when the active tab changes.    |
| `onTabClick`    | `(value: string) => void`              | `undefined` | Callback function when a tab is clicked.          |
| `size`          | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`      | Controls the size of the tab buttons.             |
