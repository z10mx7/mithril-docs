# Sidebar Component Documentation

The **Sidebar** component is a flexible and customizable navigation panel. It supports collapsible menus, nested links, and icons, making it ideal for documentation sites, dashboards, and applications with structured navigation.

## Props

### `<SidebarProvider>`

| Prop                | Type            | Default   | Description |
|---------------------|----------------|-----------|-------------|
| children           | React.ReactNode | -         | The child components of the provider. |
| defaultOpen        | boolean         | true      | Determines if the sidebar is open by default. |
| defaultSide        | 'left' \| 'right' | 'left'    | The default position of the sidebar. |
| defaultMaxWidth    | number          | 280       | The maximum width of the sidebar in pixels. |
| showIconsOnCollapse | boolean         | true      | Determines if icons should be shown when the sidebar is collapsed. |
| mobileView        | boolean         | true      | Enables mobile-specific sidebar behavior. |

### `<Sidebar>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| className  | string            | -       | Additional CSS classes for styling. |
| children   | React.ReactNode   | -       | The child elements inside the sidebar. |

### `<SidebarHeader>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The header content of the sidebar. |

### `<SidebarContent>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The main content area of the sidebar. |

### `<SidebarFooter>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The footer section of the sidebar. |

### `<SidebarMenu>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The navigation menu inside the sidebar. |

### `<SidebarMenuItem>`

| Prop          | Type              | Default    | Description |
|--------------|-------------------|------------|-------------|
| icon         | React.ReactNode   | -          | An optional icon to display in the menu item. |
| label        | string            | -          | The text label for the menu item. |
| href         | string            | -          | The link to navigate to. |
| children     | React.ReactNode   | -          | Sub-menu items if applicable. |
| isActive     | boolean           | undefined  | Manually set active state. |
| defaultOpen  | boolean           | false      | Whether the submenu is open by default. |
| alwaysOpen   | boolean           | false      | Keeps the submenu permanently open. |
| isCollapsable | boolean           | false      | Whether the submenu can be collapsed. |

### `<SidebarLayout>`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The child components inside the layout. |

### `<MainContent >`

| Prop       | Type              | Default | Description |
|------------|-------------------|---------|-------------|
| children   | React.ReactNode   | -       | The main content area. |
