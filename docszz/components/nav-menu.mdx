# NavMenu

The `NavMenu` component is a flexible and customizable navigation menu that supports dynamic positioning based on screen space.

## Props

### `NavMenu`

| Prop       | Type  | Default | Description |
| ---------- | ----- | ------- | ----------- |
| `position` | `up` | `down` | `'down'`    |

### `NavMenuItem`

| Prop        | Type         | Default | Description                                        |
| ----------- | ------------ | ------- | -------------------------------------------------- |
| `onClick`   | `() => void` | `-`     | Function to execute when the menu item is clicked. |
| `className` | `string`     | `-`     | Additional class names for styling.                |

## Positioning Behavior

The `position` prop controls whether the menu expands upwards or downwards. If not specified, the menu dynamically adjusts based on available screen space.

### Example with `position="up"`

```tsx
<NavMenu position="up">
  <NavMenuItem>Settings</NavMenuItem>
  <NavMenuItem>Logout</NavMenuItem>
</NavMenu>
```

## Custom Styling

You can extend the `NavMenu` styles using Tailwind CSS or custom class names.

```tsx
<NavMenu className="bg-gray-800 text-white">
  <NavMenuItem className="hover:bg-gray-700">Profile</NavMenuItem>
</NavMenu>
```

## Accessibility

- `aria-expanded` is automatically managed for toggling the menu.
- Uses `role="menu"` and `role="menuitem"` for proper navigation.
