# Popover Component

The `Popover` component provides a flexible and responsive floating container that can be triggered to appear around any element.

## Features

- **Smart Positioning**: Automatically adjusts to available space.
- **Click Outside Detection**: Closes when clicking outside the popover.
- **Responsive**: Adapts to different screen sizes.
- **Customizable**: Accepts any content inside.

## Props

### `Popover`

| Prop           | Type                      | Default     | Description                                 |
| -------------- | ------------------------- | ----------- | ------------------------------------------- |
| `open`         | `boolean`                 | `false`     | Controls whether the popover is open.       |
| `onOpenChange` | `(open: boolean) => void` | `undefined` | Callback fired when the open state changes. |

### `PopoverContent`

| Prop         | Type                                 | Default      | Description                                           |
| ------------ | ------------------------------------ | ------------ | ----------------------------------------------------- | 
| `side`       | `"top" \| "bottom" \|"left"\| "right"`| `"bottom"`   | Determines the preferred position of the popover. |
| `align`      | `"start"\|"center"\|"end"`         | `"center"`   | Controls the alignment of the popover relative to the trigger. |
| `sideOffset` | `number`                             | `4`          | Sets the spacing between the popover and the trigger. |

## Usage

```tsx
<Popover>
  <PopoverTrigger>
    <button>Click me</button>
  </PopoverTrigger>
  <PopoverContent>
    <p>Popover content here.</p>
  </PopoverContent>
</Popover>
```

### With Close Button

```tsx
<Popover>
  <PopoverTrigger>
    <button>Open</button>
  </PopoverTrigger>
  <PopoverContent>
    <p>Some content.</p>
    <PopoverClose>
      <button>Close</button>
    </PopoverClose>
  </PopoverContent>
</Popover>
```
