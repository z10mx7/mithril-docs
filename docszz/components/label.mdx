# Label Component Documentation

The **Label** component is a lightweight, theme-aware, and variant-powered text label that can be used with form controls like inputs, checkboxes, radio buttons, and more. It's built using **Tailwind CSS** and **class-variance-authority (CVA)** for scalable styling logic.

## Styling

The component uses `class-variance-authority` to manage label style variants:

```ts
const labelVariants = cva("text-sm font-medium leading-none", {
  variants: {
    intent: {
      default: "text-gray-900 dark:text-gray-100",
      muted: "text-gray-500 dark:text-gray-400",
      error: "text-red-600 dark:text-red-400",
    },
  },
  defaultVariants: {
    intent: "default",
  },
});
```

You can customize these classes to match your design system.

## Usage Example

```tsx
<Label htmlFor="acceptTerms" intent="muted">
  Accept terms and conditions
</Label>
<Checkbox id="acceptTerms" />
```
