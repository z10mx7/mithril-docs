# Installation

This guide installs the Mithril CLI and sets up a new project.

## Prerequisites

- **Go 1.24+** — [Download Go](https://go.dev/dl/)
- **Git** — [Download Git](https://git-scm.com/downloads)
- **Make** — GNU Make (pre-installed on macOS/Linux)
- **Docker** (optional) — for local Postgres and dev services

## Install the CLI

### Method 1: Install script (recommended)

```bash
curl -fsSL https://raw.githubusercontent.com/mithril-framework/mithril/main/install.sh | sh
```

This runs `go install github.com/mithril-framework/mithril/cmd/mithril@latest` and prints next steps.

Ensure `$(go env GOPATH)/bin` is on your `PATH`.

### Method 2: go install

```bash
go install github.com/mithril-framework/mithril/cmd/mithril@latest
mithril --version
```

### Method 3: From source

```bash
git clone https://github.com/mithril-framework/mithril.git
cd mithril
go build -o bin/mithril ./cmd/mithril
sudo ln -sf "$(pwd)/bin/mithril" /usr/local/bin/mithril
mithril --version
```

## Create a Project

```bash
mithril new my-awesome-app
cd my-awesome-app
cp env.example .env
```

`mithril new` clones the framework template, removes `.git`, copies `env.example` to `.env`, and runs `go mod tidy`.

## Start Developing

```bash
# Start PostgreSQL (Docker)
make dc-up-postgres

# Run migrations
mithril migrate-up

# Create admin user
mithril createsuperuser

# Run server (port 4000)
mithril run
```

Visit:

- App: http://localhost:4000
- Swagger: http://localhost:4000/docs
- Health: http://localhost:4000/health

## Clone the Framework Repo

To contribute or run the framework itself:

```bash
git clone https://github.com/mithril-framework/mithril.git
cd mithril
make install
cp env.example .env
make dc-up-postgres
make migrate-up
make run
```

Inside a project, `mithril <target>` delegates to `make` (same as `./mithril` shell script).

## Environment Variables

Copy `env.example` to `.env`:

```bash
cp env.example .env
```

Key variables:

| Variable | Description |
|----------|-------------|
| `PORT` | HTTP port (default `4000`) |
| `JWT_SECRET` | **Required** when `APP_ENV=production` |
| `DATABASE_URL` | PostgreSQL connection string |
| `ENABLE_REGISTER` | Set `true` to allow `POST /auth/register` |
| `ENABLE_COMPRESSION` | Enable gzip responses |
| `ENABLE_HELMET` | Security headers |
| `CORS_ORIGINS` | Comma-separated allowed origins |

```
├── pkg/utils/              # Shared utilities
├── public/admin/           # Admin SPA
├── infrastructure/         # Docker, K8s, compose
├── Makefile

```

## Troubleshooting

**`mithril: command not found`**

Add Go bin to PATH:

```bash
export PATH="$(go env GOPATH)/bin:$PATH"
```

**Database connection failed**

Ensure Postgres is running and `DATABASE_URL` in `.env` matches your setup. With Docker:

```bash
make dc-up-postgres
# Default DB: mithril_rev on localhost:5432
```

**Go version mismatch**

Mithril requires Go 1.24+. Check with `go version`.
