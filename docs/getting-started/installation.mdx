# Installation

This guide will help you install the Mithril framework and get started with your first project.

## Prerequisites

Before installing Mithril, make sure you have the following installed:

- **Go 1.21 or later** - [Download Go](https://golang.org/dl/)
- **Git** - [Download Git](https://git-scm.com/downloads)
- **Make** - [GNU Make](https://www.gnu.org/software/make/)
- **Docker** (optional) - [Download Docker](https://www.docker.com/get-started)
- **PostgreSQL** (optional, for local development) - [Download PostgreSQL](https://www.postgresql.org/download/)

## Installation Methods

### Method 1: Universal One-Line Installation (Recommended)

```bash
# Automatic OS detection and installation
curl -fsSL https://raw.githubusercontent.com/mithril-framework/mithril/main/install.sh | sh
```

This single command automatically:

- Detects your operating system (Linux, macOS, Windows)
- Downloads the appropriate installation script
- Installs Mithril with the best method for your platform
- Verifies the installation

### Method 2: Manual Installation

```bash
# Install the global CLI tool manually
go install github.com/mithril-framework/mithril/cmd/mithril@latest

# Verify installation
mithril --version
```

### Method 3: Install from Source

```bash
# Clone the repository
git clone https://github.com/mithril-framework/mithril.git
cd mithril

# Build and install
go build -o mithril cmd/mithril/main.go
sudo mv mithril /usr/local/bin/

# Verify installation
mithril --version
```

### Method 4: Using Go Modules (For Development)

```bash
# Create a new directory for your project
mkdir my-mithril-project
cd my-mithril-project

# Initialize Go module
go mod init my-mithril-project

# Add Mithril as a dependency
go get github.com/mithril-framework/mithril@latest

# Create a simple main.go
cat > main.go << 'EOF'
package main

import (
    "log"
    "github.com/mithril-framework/mithril/pkg/core"
)

func main() {
    app := core.NewApplication()
    log.Fatal(app.Listen(":3000"))
}
EOF

# Run the application
go run main.go
```

## Installation Script Explained

The Mithril framework provides a universal installation script that automatically detects your operating system and installs the appropriate version:

### Universal Installation Script (`install.sh`)

- **What it does**: Automatically detects your OS and installs Mithril using the best method for your platform
- **Platforms**: Linux, macOS, Windows (with WSL), and other Unix-like systems
- **Requirements**:
  - `curl` or `wget` for downloading
  - Go 1.21+ (for fallback building from source)
- **Features**:
  - Automatic OS detection
  - Downloads pre-built binaries when available
  - Falls back to building from source if needed
  - Verifies installation after completion
  - Provides helpful error messages and troubleshooting tips

## Quick Start

### 1. Create a New Project

```bash
# Create a new Mithril project
mithril new my-awesome-app

# Navigate to the project directory
cd my-awesome-app

# Install dependencies
mithril install

# or use shorthand
mithril i
```

### 2. Configure Environment

```bash
# Copy the example environment file
mv env.example .env

# Edit the environment file
vi .env

# make sure do all needed changes in .env file ,  including database credentials and jwt secret and apikey's
```

### 3. Set Up Database

```bash
# Run database migrations (GORM AutoMigrate)
# Make sure your models are registered using migration.RegisterModel() in their init() functions
mithril migrate-up

# Seed the database (optional)
mithril seed
```

### 4. Start Development Server

```bash
# Start the development server with live reload
mithril run-dev

# Or start with docker
mithril run-docker
```

### 5. Access Your Application

- **Web Application**: http://localhost:4000
- **API Documentation**: http://localhost:4000/docs
- **Health Check**: http://localhost:4000/health
- **Live Check**: http://localhost:4000/livez
- **Ready Check**: http://localhost:4000/readyz
- **Resource and Usage Monitor**: http://localhost:4000/monitor
- **Admin Panel (need a command to run)**: http://localhost:4000/admin
- **DBMS Panel (need a command to run)**: http://localhost:5050

## Configuration

### Environment Variables

Mithril uses environment variables for configuration. Copy `env.example` to `.env` and customize:

```bash
PORT=4000
APP_NAME=mithril
# Process timezone (IANA, e.g. America/New_York). APP_TIMEZONE overrides TZ when both are set.
TZ=UTC
JWT_SECRET=generate-a-secret-key-for-your-application

# PostgreSQL (optional; if unset, app runs without DB)
DATABASE_URL=postgres://user:password@localhost:5432/mithril?sslmode=disable

# Or use separate vars (app and make migrate-* use DATABASE_URL when set):
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=""
DB_NAME=mithril
DB_SSLMODE=disable

# Set to 1 to allow POST /auth/register; when unset, registration is disabled (app cannot create users via API).
ENABLE_REGISTER=true

# Set to true to enable gzip response compression.
ENABLE_COMPRESSION=true

# Set to true to enable Helmet security headers (XSS, nosniff, X-Frame-Options, etc.).
ENABLE_HELMET=true

OPENAI_API_KEY=

# Embedded PostgreSQL web UI (Adminer-style): ENABLE_DBMS=true or touch .dbms-enabled; then `make dbms`.
# Default listen address :5050; override with DBMS_ADDR=:9090 (legacy: ADDR is also read if DBMS_ADDR is empty).
ENABLE_DBMS=true
DBMS_ADDR=:5050
```

## Troubleshooting

### Common Issues

#### 1. Installation Script Fails

```
curl: command not found
```

**Solution**:

- Install curl: `sudo apt-get install curl` (Ubuntu/Debian) or `brew install curl` (macOS)
- Use manual installation method: `go install github.com/mithril-framework/mithril/cmd/mithril@latest`
- Check your internet connection

#### 2. Go Version Error

```
go: requires go >= 1.21
```

**Solution**: Update Go to version 1.21 or later.

#### 3. Binary Not Found After Installation

```
mithril: command not found
```

**Solution**:

- Restart your terminal
- Check if `/usr/local/bin` is in your PATH
- Try manual installation: `go install github.com/mithril-framework/mithril/cmd/mithril@latest`

#### 4. Database Connection Error

```
failed to connect to database
```

**Solution**:

- Check if the database server is running
- Verify connection details in `.env`
- Ensure the database exists

#### 5. Port Already in Use

```
bind: address already in use
```

**Solution**:

- Kill the process using the port: `mithril kill` and start mithril again using `mithril run-dev`
- Use a different port in `.env` file and start mithril again using `mithril run-dev`
- Kill the process using the port: `lsof -ti:3000 | xargs kill -9`

#### 6. Permission Denied

```
permission denied
```

**Solution**:

- Use `sudo` for global installation
- Check file permissions
- Ensure you have write access to the directory

## Uninstallation

### Remove Global CLI

```bash
# If installed via go install
go clean -i github.com/mithril-framework/mithril/cmd/mithril

# If installed manually
sudo rm /usr/local/bin/mithril
```

### Remove from Go Module

```bash
# Remove from go.mod
go mod edit -droprequire github.com/mithril-framework/mithril

# Clean module cache
go clean -modcache
```
