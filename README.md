# Mithril Documentation

Documentation site for the [Mithril](https://github.com/mithril-framework/mithril) Go web framework.

**Live site:** [mithril-docs-nine.vercel.app](https://mithril-docs-nine.vercel.app)

## Stack

- Next.js 15 + Contentlayer2 + MDX
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Contributing

Add or edit MDX files under `docs/`. Update `config/sidebar.tsx` for navigation. Run `npm run build` to verify Contentlayer compiles.

Framework source: [github.com/mithril-framework/mithril](https://github.com/mithril-framework/mithril)
